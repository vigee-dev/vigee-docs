# Démarrage rapide

Bienvenue dans la documentation Vigee ! Ce guide vous aidera à démarrer rapidement.

## Prérequis

- Node.js 18+
- PHP 8.2+
- pnpm
- Composer
- Docker (optionnel)

## Installation

```bash
# Cloner le monorepo
git clone git@github.com:votre-org/vigee.git
cd vigee

# Installer les dépendances
pnpm install
composer install
```
