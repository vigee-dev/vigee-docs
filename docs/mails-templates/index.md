# Templates d'emails

Bibliothèque de templates d'emails prêts à l'emploi pour la communication client.

::: tip Comment utiliser
Cliquez sur le bouton **Copier** en haut à droite de chaque bloc pour copier le texte.
Remplacez ensuite les éléments entre crochets `[...]` par vos informations.
:::
