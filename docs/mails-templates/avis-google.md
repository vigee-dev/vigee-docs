# Demande d'avis Google

Template pour solliciter un avis client sur Google.

## Objet

```
Votre retour sur notre collaboration
```

## Contenu

```
Bonjour,

Votre retour sur notre collaboration compte réellement pour nous. Il nous permet d'évaluer la qualité du travail accompli et de continuer à maintenir un niveau d'exigence élevé.

Si tu es d'accord, tu peux laisser un avis ici : https://g.page/r/CbBoQ9rdFoP1EBM/review

C'est rapide, quelques lignes suffisent.

Merci d'avance pour ton retour.

Bien à toi,
Romain
```

## Contexte d'utilisation

À envoyer après :
- Une livraison réussie
- La fin d'un projet
- Un retour positif spontané du client

::: tip Timing
Idéalement, envoyer cet email quelques jours après une interaction positive avec le client.
:::
