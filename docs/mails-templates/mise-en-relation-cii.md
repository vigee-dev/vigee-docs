# Mise en relation - Crédit Impôt Innovation

Template pour mettre en relation un client avec un expert CII.

## Objet

```
Mise en relation - Agrément CII
```

## Contenu

```
Bonjour,

Je me permets de vous mettre en relation suite à l'obtention par Vigee du Crédit Impôt Innovation.

L'objectif serait d'en faire bénéficier [NOM_SOCIETE] pour le développement de l'application sur mesure développée par Vigee.

@[PRENOM_EXPERT], vous trouverez en destinataire [PRENOM_CLIENT] [NOM_CLIENT], [POSTE_CLIENT] de [NOM_SOCIETE].

Je reste à votre écoute pour tout besoin complémentaire,

À bientôt,
Romain
```

## Variables à remplacer

| Variable | Description | Exemple |
|----------|-------------|---------|
| `[NOM_SOCIETE]` | Nom de la société cliente | ACME Corp |
| `[PRENOM_EXPERT]` | Prénom de l'expert CII | Julien |
| `[PRENOM_CLIENT]` | Prénom du client | Pierre |
| `[NOM_CLIENT]` | Nom du client | Dupont |
| `[POSTE_CLIENT]` | Poste du client | directeur |

## Contexte

Le Crédit Impôt Innovation (CII) permet aux PME de bénéficier d'un crédit d'impôt de 20% des dépenses engagées pour la conception de prototypes ou installations pilotes de produits nouveaux.

::: info Vigee et le CII
Vigee dispose de l'agrément CII, ce qui permet à ses clients de valoriser les développements réalisés dans le cadre de ce dispositif fiscal.
:::
