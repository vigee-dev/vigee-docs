# PV de livraison

Template pour demander la signature du Procès-Verbal de Livraison.

## Objet

```
Signature du Procès-Verbal de Livraison
```

## Contenu

```
Bonjour,

Je te transmets ci-joint le Procès-Verbal de Livraison relatif au module [MODULE] que nous venons d'achever.

Tu recevras également la demande de signature électronique via Yousign.

Lorsque tu auras pris connaissance du document, tu pourras procéder à la signature si tout est conforme pour toi.

Et si tu souhaites échanger sur un point avant validation, je suis bien entendu disponible.

Bien à toi,
Romain
```

## Variables à remplacer

| Variable | Description |
|----------|-------------|
| `[MODULE]` | Nom du module livré (ex: "réservation de salles") |

## Pièces jointes

- PV de livraison (PDF)
- Envoi Yousign séparé pour signature électronique
