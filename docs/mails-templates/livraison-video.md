# Livraison vidéo

Template pour l'envoi d'une vidéo de présentation des développements livrés.

## Objet

```
Livraison des développements
```

## Contenu

```
Bonjour [PRENOM],

Je suis ravi de vous partager la vidéo de présentation des nouvelles fonctionnalités que nous avons développées.

Vous pouvez accéder à la vidéo via le lien suivant : [Lien vers la vidéo].

Afin de garantir que tout fonctionne selon vos attentes, je vous invite à réaliser les tests de votre côté. Cela nous permettra de valider ensemble que l'ensemble des fonctionnalités répond bien à vos besoins avant la mise en production.

N'hésitez pas à nous faire un retour une fois les tests réalisés, que ce soit pour nous signaler d'éventuels ajustements à apporter ou pour valider l'ensemble.

Nous restons bien entendu à votre disposition si vous avez la moindre question ou si vous souhaitez un accompagnement lors de ces tests.

Dans l'attente de vos retours,

Bien cordialement,
Romain
```

## Variables à remplacer

| Variable | Description |
|----------|-------------|
| `[PRENOM]` | Prénom du destinataire |
| `[Lien vers la vidéo]` | URL de la vidéo (Loom, YouTube, etc.) |
