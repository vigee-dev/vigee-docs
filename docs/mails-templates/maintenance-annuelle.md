# Maintenance annuelle - Explication

Template pour expliquer le détail du forfait de maintenance mensuel.

## Objet

```
Détail du forfait de maintenance mensuel
```

## Contenu

```
Bonjour [PRENOM],

Suite à nos précédents échanges, voici le détail du forfait de maintenance mensuel que nous mettons en place pour votre plateforme (back-office web, application mobile, et API). Ce forfait est aligné avec les standards des applications en production à fort usage, et tient compte de votre rythme de croissance actuel.

Le volume mensuel est désormais fixé à [NOMBRE] heures réparties comme suit :

┌─────────────────────────────┬────────────────────────────────────────────────────────────────────────────┬─────────────────┐
│ Poste                       │ Contenu couvert                                                            │ Volume mensuel  │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Maintenance technique       │ Supervision serveur, sauvegardes, mises à jour de sécurité,               │ 2 h             │
│                             │ montée de version des dépendances (Next.js, Laravel, SDK mobiles, etc.)   │                 │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Correctifs Web & Back-office│ Résolution de bugs, micro-évolutions fonctionnelles                       │ 2 h             │
│                             │ et ajustements UI/UX côté administration                                  │                 │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Correctifs Mobile           │ Suivi des crashs iOS/Android, mises à jour store,                         │ 1 h             │
│                             │ adaptations UI multi-devices                                              │                 │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Assistance & support        │ Réponse aux tickets via Vigee Flow, échanges mail/téléphone,              │ 3 h             │
│ (web + mobile)              │ accompagnement utilisateurs                                               │                 │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Suivi & pilotage            │ Priorisation des demandes, coordination des deux développeurs dédiés,     │ 2 h             │
│                             │ reporting mensuel                                                         │                 │
├─────────────────────────────┼────────────────────────────────────────────────────────────────────────────┼─────────────────┤
│ Total mensuel               │                                                                            │ 10 h            │
└─────────────────────────────┴────────────────────────────────────────────────────────────────────────────┴─────────────────┘

Points clés :

• Traction élevée : ce volume garantit le même niveau de réactivité malgré l'augmentation continue des utilisateurs prévue
• Couverture complète : une seule enveloppe pour web et mobile afin de simplifier la gestion et éviter les dépassements
• Maintien en conditions opérationnelles : toutes les refontes techniques imposées par les évolutions des frameworks sont incluses
• Revue annuelle : bilan chaque fin d'année pour ajuster si nécessaire, à la hausse ou à la baisse, en fonction de vos usages réels

Je reste à ta disposition pour toute question avant l'envoi du contrat formel.

À bientôt,
Romain
```

## Variables à remplacer

| Variable | Description |
|----------|-------------|
| `[PRENOM]` | Prénom du destinataire |
| `[NOMBRE]` | Nombre d'heures du forfait (ex: 10) |

## Notes

- Adapter les heures par poste selon le contrat client
- Supprimer la ligne "Correctifs Mobile" si pas d'application mobile
