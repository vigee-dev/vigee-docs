# Suivi des développements

Template pour informer le client de l'avancement des développements.

## Objet

```
Suivi des développements
```

::: info CC
Mettre en copie : romain et robin
:::

## Contenu

```
Bonjour [PRENOM],

Les développements du module [MODULE] sont en cours, et nous avons établi le planning prévisionnel suivant :

• Phase de développement & tests internes : [DATES]
• Livraison estimée sur le serveur de développement : [DEADLINE]

À l'issue de nos tests internes, vous aurez accès :

• à une vidéo de présentation des nouveaux éléments
• au serveur de développement pour tester directement les nouvelles fonctionnalités

Nous restons bien entendu à votre disposition pour toute information complémentaire.

Bien cordialement,
Romain
```

## Variables à remplacer

| Variable | Description |
|----------|-------------|
| `[PRENOM]` | Prénom du destinataire |
| `[MODULE]` | Nom du module en développement |
| `[DATES]` | Période de développement (ex: "du 15 au 30 janvier") |
| `[DEADLINE]` | Date de livraison estimée |
